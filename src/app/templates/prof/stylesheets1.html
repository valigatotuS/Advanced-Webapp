Stylesheets toegepast

Bekijk de broncode van deze webpagine. Bovenaan de pagina in het head gedeelte vind je volgende verwijzing:

<link rel="stylesheet" href="stylesheets1.css">

Door toewijzing van één of meerdere classen en/of één id attribuut aan HTML elementen kunnen verschillende CSS entries aangesproken worden.

De CSS stijl wordt gespecifieerd op een volledig HTML type (vb h1), op een bepaalde class (vb .emph, .htmltags), of op een id (vb #stylesheetlink).
Het is ook mogelijk om een cascade van selectoren te maken waarop de specifieke stijl toepassing heeft (vb ul.blokjes, ul.blokjes li en ul.blokjes li .htmltags).
Toegepaste stijlen worden verder doorgegeven aan alle kind elementen waarop ze toepassing hebben.

Kijk naar de HTML en CSS broncode om te zien hoe dit in zijn werk gaat.

HTML staat voor HyperText Markup Language. Het is geen programmeertaal maar een opmaaktaal zoals bijvooreeld LaTeX.

HTML elementen zijn steeds geplaatst aan de hand van HTML tags, geplaatst binnen <...> code. Het merendeel van de HTML elementen moet ook opnieuw gesloten worden via de bijhorende </...> tag

Enkele voorbeelden van courante HTML elementen:

  • <p> een paragraaf </p>
  • <h1> headings/titels met grootte gaande van 1 (grootst) tot 6 </h1>
  • <a href="https://www.vub.be"> een hyperlink naar https://www.vub.be met dit als tekst</a>
  • <img src="mijnafbeelding.png" width="200px">
  • <br> 1 lijn line break
  • <u> onderlijnen </u>
  • <b> vetgedrukt </b>
  • <i> italic </i>
  • <button> klikopmij</button>

HTML elementen bestaan dus steeds uit een HTML openingstag, eventuele HTML attributen (zoals href="https://www.vub.be" of width="200px"), eventuele inhoud (dit kan tekst zijn maar kan ook andere HTML elementen bevatten), en afhankelijk van de gebruikte HTML tag ook een sluitingstag

HTML maakt ook gebruik van onder meer de <html>, <head>, <title> en <body> tags om de metadata van de webpagina te vormen.
